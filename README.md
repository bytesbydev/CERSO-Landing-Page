# 🚀 Cerso Landing Page

A modern, responsive landing page built for **Cerso**, a software development studio that helps startups, businesses, and organizations launch professional websites in days—not months.

Built with **React**, **Vite**, **Tailwind CSS**, and **Lucide React**.

---

## ✨ Features

- 🎨 Modern SaaS-inspired UI
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎯 Smooth Scroll Navigation
- 💼 Services Section
- 💰 Pricing Plans
- 🔄 Process Timeline
- ⭐ Testimonials
- ❓ FAQ Section
- 📩 Contact Form
- 🌙 Clean & Minimal Design

---

## 🛠 Tech Stack

- React 19
- Vite
- Tailwind CSS
- Lucide React
- JavaScript (ES6+)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/cerso-landing.git
```

### 2. Navigate to the project

```bash
cd cerso-landing
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
